# Table Monitor - Мониторинг занятости стола

Система компьютерного зрения для автоматического мониторинга занятости столов с использованием YOLOv8. Программа анализирует видеопоток, определяет когда стол свободен или занят, сохраняет статистику и логирует все события.

## Возможности

### Основные функции

- **Автоматическая детекция людей** с использованием YOLOv8
- **Интерактивный выбор области стола** (ROI) мышкой
- **Масштабирование окна** с сохранением координат (от 30% до 200%)
- **Фильтрация ложных срабатываний** через порог стабильности
- **Логирование всех событий** в реальном времени

### Навигация по видео

- **Трекбар** для быстрой перемотки в любую точку
- **Клавиши перемотки** вперед/назад (30 секунд)
- **Клавиши Home/End** для перехода в начало/конец
- **Пауза/продолжение** для детального анализа

### Статистика и аналитика

- Время ожидания клиентов (среднее, медиана, мин/макс)
- Количество событий (занят/свободен)
- Производительность детекции (время обработки кадра)
- Экспорт статистики в CSV

### Логирование событий

- Детальное логирование всех действий в файл
- Отдельный файл для событий
- Ротация лог-файлов (10MB, 5 файлов)

## Установка

### Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск программы

```bash
python main.py --video название_видео.mp4
```

При первом запуске программа покажет первый кадр видео:
1.Выделите область стола с помощью мыши;
2.Нажмите Enter/Пробел для подтверждения.

После выбора области начнётся анализ видео:
1.Зелёный прямоугольник = стол свободен;
2.Красный прямоугольник = стол занят;
3.Статистика отображается в левом верхнем углу.

## Проблемы и решения

### 1.Плохая детекция людей спиной

Проблема: YOLOv8n плохо детектирует людей, сидящих спиной к камере.

Решение:

- Использование порога стабильности (3 кадра)
- Детекция по движению в области

### 2.Ложные срабатывания

Проблема: Проходящие мимо люди могут вызвать ложное срабатывание.

Решение:

- Порог стабильности 3 кадра
- Фильтрация по уверенности (0.25)
- Оптимальный выбор ROI
